# DevOps Roadmap

## Stage 1: Foundations

- Linux filesystem and permissions
- networking basics
- Git branching and remote workflows
- YAML syntax

## Stage 2: CI/CD

- Jenkins installation
- freestyle jobs
- declarative pipelines
- credentials and agents
- SonarQube quality gates

## Stage 3: Build and Deploy

- Maven lifecycle
- artifact packaging
- Tomcat deployment
- rollback strategy

## Stage 4: Containers

- Dockerfile design
- image layers
- container networking
- registry push/pull

## Stage 5: Kubernetes

- pods
- deployments
- services
- config maps and secrets
- local clusters with KIND

## Stage 6: Infrastructure as Code

- Terraform init/plan/apply
- variables and outputs
- remote state
- modules
- AWS and Azure providers

## Stage 7: Production Discipline

- least privilege IAM
- secret management
- observability
- cost controls
- disaster recovery basics
